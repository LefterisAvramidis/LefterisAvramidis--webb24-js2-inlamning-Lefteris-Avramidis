Start instruktioner till mitt projekt!

---STARTA SERVER---------------------------------------------------
Navigera till rätt fil med hjälp av cd på följande sätt:
1. cd "GN - Projekt"
2. cd Backend
3. cd src

Därefter skriver du "node server.js" för att starta node servern. 
-------------------------------------------------------------------

---STARTA HTML--------------------------------------------------------------------
Jag använder Visual Studio Code-tillägget "Live Server". Högerklicka på index.html 
och välj "Open with Live Server" för att öppna filen i din webbläsare.
----------------------------------------------------------------------------------

---HIGHSCORE LISTA------------------------------------------------------------------------------------
Highscore-listan uppdateras i realtid. Varje spelomgång sparas och rankas från högst till lägst poäng. 
Det finns även en knapp för att nollställa listan om du vill börja om från början.
